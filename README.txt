Project: Athens Eats
Functionality:
	This project allows a logged in user to rate and leave
	reviews on restaurants located in athens.
Starting project: 
1. open project folder athens-eats-main
2. import atheanseats_db.sql
3. give the user "springuser" the password "ThePassword"
4. open file athens-eats-main/src/main/resources/application.properties
5. comment out line 6 by typing # at the front. save file 
6. run athens-eats-main/src/main/java/com/athenseats/server/AthensEatsApplication.java
7. open the application.properties file and uncomment line 6 and save.
8. re-run AthensEatsApplication.java. database should now be populated.
9. open terminal navigate to athens-eats-main/src/main/java/com/athenseats/client directory
10. run these commands:
	npm install
	npm start
11. project should run.

Project has been tested on:
	os: Windows 10, MacOS
	browsers: Chrome, FireFox
